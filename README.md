# Next.js App Router Blog Uygulaması

Bu proje, Next.js App Router kullanarak oluşturulmuş basit bir kişisel blog uygulamasıdır. Markdown (MDX) formatında yazılmış blog yazılarını okuyarak dinamik olarak oluşturur ve görüntüler.

## Özellikler

- **Next.js App Router:** Next.js'in yeni rotalama sistemi olan App Router ile geliştirilmiştir.
- **MDX Desteği:** Blog yazıları Markdown (MDX) formatında yazılır, bu sayede kolayca biçimlendirilebilir ve React bileşenleri ile zenginleştirilebilir.
- **Dinamik Rotalar:** Her blog yazısı için otomatik olarak dinamik rotalar oluşturulur.
- **Ön Bilgiler (Front Matter):** Blog yazılarının başlık, tarih gibi bilgileri front matter ile yönetilir.
- **TypeScript:** Proje TypeScript ile yazılmıştır, bu sayede daha güvenli ve ölçeklenebilir bir kod tabanı elde edilir.
